## Project Overview
This project analyzes which team statistics most strongly influenced AP Top 25 rankings in the 2022 college football season.

## Data Sources
- AP Top 25 rankings dataset
- College football team statistics dataset

## Methods
- Data cleaning
- Dataset merging
- Correlation analysis

## Key Findings
- Offensive stats (points per game, wins) had strongest relationship
- Defensive stats were weaker predictors

## Files Included
- Notebook (.ipynb)
- Data (.csv)
- Final report (.pdf)
