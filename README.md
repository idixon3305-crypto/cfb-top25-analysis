# College Football Top 25 Analysis (2022)

## Overview
This project analyzes which team statistics are most strongly related to final AP Top 25 rankings in the 2022 college football season.

## Method
Correlation analysis was used to evaluate the strength of relationships between team performance metrics and AP ranking.

## Key Findings
- Offensive performance (points per game, wins, efficiency) had the strongest relationship with ranking
- Defensive statistics showed weaker relationships
- Higher-ranked teams consistently scored more and won more games

## Files
- Notebook: cfb_top25_analysis.ipynb
- Data: CSV files
